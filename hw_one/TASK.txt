0. Общие требования
- Оформить в виде консольного приложения
- Ввод данных через аргументы или интерактивный режим (лучше использовать оба в разных задачах)

1. Узнать в какой день недели ты родился по дате своего Дня Рождения
- Результат выводится на русском или английском, а не просто цифрой
- Нельзя использовать модули date и datetime
- Не забыть про нестандартные ситуации
- Должно работать для любого Дня Рождения, то есть для любой даты

2. Простая математика
Есть три простые задачи, как минимум нужно реализовать одну, но лучше все:
- Наименьшее общее кратное
- Наибольший общий делитель
- Список простых чисел до заданного значения
Если есть вопросы по тому, что это такое и как должно считаться - эта информация есть в википедии
Без использования внешних модулей (ни одного импорта в коде).

3. Математика посложнее
Опять же минимум одна задача, лучше все:
- Факториал
- Ряд Фибоначчи (до заданного значения)
- Квадратный корень (с точностью минимум 2 знака после запятой)
Если есть вопросы по тому, что это такое и как должно считаться - эта информация есть в википедии
Без использования внешних модулей (ни одного импорта в коде).
